# Project name: The Weather Dashboard

## About the Project

The Weather Dashboard is a CLI application that fetches real-time weather data from the Open-Meteo API to display formatted forecasts and maintain a local weather history. It demonstrates essential programming skills such as asynchronous API integration, HTTP request handling, and formatted console output.

## Technologies

- **Node.js** - JavaScript runtime
- **VS Code** - Code editor
- **NPM** - Package manager

## Installation

Download the repository:

```bash
git clone https://github.com/yourusername/projectname.git
```

Navigate to the project folder:

```bash
cd \path\of\my\projectname
```

Install the dependencies:

```bash
npm install
```

## Start project

Run the application using Node.js:

```bash
node index.js
```
